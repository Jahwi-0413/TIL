## Babel vs SWC

---

- Babel이란?
  - javascript compiler
  - transpiler
    - 유사한 두 언어 사이에서 변환 제공
    - 최신 javascript 문법을 구버전 브라우저에서도 사용할 수 있도록 변환해줌
  - polyfill → babel이 변환해주지 않는 문법을 지원하기 위한 모듈
- SWC
  - javascript compiler
  - babel보다 빌드 시간이 최대 5배 빠름
